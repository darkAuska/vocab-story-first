# 背诵闭环

## 每天不是“读完”，而是完成四个动作

1. **预检**：先看今日 200 词，标出完全不认识的词。
2. **语境输入**：读游戏对话正文，只关注目标词在剧情里的作用。
3. **主动回忆**：做英译中、中译英、完形填空，不看正文。
4. **错词登记**：把不会、反应慢、拼错、熟词僻义的词写入 `状态/错词记录.csv`。

## 错词复习间隔

建议复习节点：

- D+1：第二天快速复现。
- D+3：三天后做中译英。
- D+7：一周后放入新剧情或短对话。
- D+14：两周后做拼写和搭配。
- D+30：一个月后做最终抽查。

## 什么算真正背住

一个词满足下面 3 条，才算暂时掌握：

- 看到英文能在 2 秒内说出核心中文含义。
- 看到中文能拼出英文，允许大小写差异，不允许核心拼写错。
- 能说出一个语境搭配或短句，而不是只记中文对照。

## 生成文件验收标准

当天 MD 文件如果缺少下面任一项，不算完成：

- 200 个目标词全部出现。
- 目标词有醒目标记。
- 有中文语义提示。
- 有主动回忆题。
- 有覆盖率自检。
- 有错词登记区。
