# 中英呈现规则

这份规则用于控制每日背单词故事的中英比例。目标是让大脑先进入英文语序，再用详细中文翻译校验理解，避免因为看不懂而跳过英文。

## 核心原则

1. **正文英文先行**：每个剧情自然段先给英文，让目标词自然嵌入英文语序和搭配。
2. **详细中文紧随其后**：英文段落后必须给详细中文翻译，帮助理解剧情、人物关系和目标词语境。
3. **CN 行中文优先**：`CN:` 行必须尽量全中文。英文目标词、英文桥段名或必要专名只能放在中文后的括号里，例如“减少、削弱（diminish）”“假想路线（if line）”。不要把英文裸露在中文句子主干里。
4. **CN 行必须是详细翻译**：`CN:` 必须逐句翻译紧挨着的上一段 `EN:`，包括动作、因果、人物情绪和对话含义。禁止把 `CN:` 写成剧情概括、词汇清单或“本段目标词包括……”。目标词释义可以嵌入译文，但不能代替译文。
5. **正文不拆成例句表**：目标词第一次出现时在英文自然段里加粗；不要一个目标词写一句话。词义、搭配、词根和词源集中放到幕后的目标词小结。
6. **场景先于单词**：先写正常场景，再把目标词自然融入环境、动作、对话、物品和心理活动。禁止写“大家的情绪因为这些词变得明显”“系统列出这些词所以剧情推进”。
7. **不要中文先行**：不要把中文放在英文前面；先读英文，再看中文翻译。
8. **背诵优先但不牺牲基本叙事**：如果剧情和背诵冲突，优先保证目标词覆盖、词义提示、主动回忆和复现，但不能把正文写成词表说明。

## 推荐正文格式

正文推荐使用“英文自然段 + 中文自然段”的结构，例如：

```md
EN: Okabe refused to **abandon** the experiment, even after the timestamp became abnormal.
CN: 冈部拒绝“放弃”（abandon）实验，即使时间戳已经变得异常。
Word: abandon = 放弃；常见搭配：abandon a plan / abandon an experiment
```

目标词第二次及以后出现时，可以不重复解释，但仍要给中文剧情翻译：

```md
EN: Kurisu warned him that abandoning the test now would erase the only clue.
CN: 红莉栖提醒他，现在放弃测试会抹掉唯一的线索。
```

错误写法：

```md
CN: 这一段写实验室成员在修复徽章时推进任务。本段目标词包括：“给；提供”（give）、“发生”（happen）。
```

这种写法只是摘要和词汇清单，不是翻译，正式文件不得使用。

## 每幕结构

每幕建议按这个顺序：

1. **Scene Title**：英文场景标题。
2. **English Story / Dialogue**：英文剧情正文，目标词高亮。每幕 3-5 个自然段，每个英文自然段后紧跟详细 `CN:` 翻译。
3. **Key Vocabulary Notes**：统一解释本幕目标词，不要在正文里逐句打断。
4. **中文剧情大意**：用中文概括本幕发生了什么，帮助复盘。
5. **本幕目标词小结**：列出目标词、中文释义、常见搭配。

## 不推荐格式

不要这样写：

```md
冈部拒绝放弃实验。
Okabe refused to abandon the experiment.
```

中文不能先于英文出现。正确顺序是 `EN -> CN -> Word`，既能看懂，又保留英文语序训练。

## 适合考研英语的加强项

- 每个目标词至少给一个常见搭配。
- 标记熟词僻义，例如 `issue = 问题；发行；发布`。
- 主动回忆题必须包含中译英，防止只会认不会拼。
- 完形填空优先抽取正文原句，训练目标词在语境中的识别。
