# 甜甜恋爱互动规则

这份规则用于增强每日背单词故事里的恋爱日常感。目标是让目标词和人物之间的暧昧、照顾、吃醋、误会、信任与小心动绑定起来，提高记忆黏性。

## 核心方向

1. **甜而不腻**：重点写细节里的好感，例如递饮料、替对方圆场、嘴硬关心、共同解谜、争吵后和好。
2. **轻喜剧暧昧**：可以写脸红、误会、吃醋、互相吐槽、被旁人起哄，但不要写露骨内容。
3. **多角色互动**：男主可以和不同女性主角、配角产生不同类型的甜互动，不要只集中在一个人身上。
4. **角色差异明确**：每位女性角色的甜感来源不同，不能都写成同一种撒娇模式。
5. **背诵优先**：恋爱互动要服务目标词记忆，目标词尽量嵌入告白未遂、关心、争论、道歉、鼓励、保护、约定等互动里。

## 互动类型建议

- **冈部 x 红莉栖**：傲娇吐槽、理性争辩、嘴硬关心、实验搭档式默契。
- **冈部 x 真由理**：青梅竹马式温柔、便当/零食/服装小日常、低压治愈、天然直觉击中要害。
- **冈部 x 铃羽**：行动派保护、并肩调查、未来战士的笨拙关心、训练和任务中的信任。
- **冈部 x 萌郁**：短信式暧昧、沉默关心、误读消息后的轻微误会、慢慢建立安全感。
- **冈部 x 菲伊丽丝**：女仆咖啡厅轻喜剧、猫耳式调侃、卡牌比赛赌约、夸张台词互相接招。
- **冈部 x 琉华**：温柔、害羞、认真练习台词、被夸奖后慌张。
- **冈部 x 真帆/篝/由季等配角**：可作为支线互动，用共同调查、研究协助、未来线索和日常照顾制造甜度。

## 每幕甜度要求

每幕至少安排一个“甜互动钩子”，例如：

- 某个角色替男主掩饰失误。
- 男主嘴上中二，行动上先保护对方。
- 女性角色发现男主其实记得她的小习惯。
- 两人因目标词含义争论，最后变成小小的心动。
- 旁人吐槽两人像在约会，当事人同时否认。
- 手机短信触发暧昧误会，但最后变成关键线索。

## 英文表达要求

恋爱互动仍然以英文正文为主。目标词第一次出现时可以使用 `EN / CN / Word` 三行解释。

示例：

```md
EN: Kurisu tried to sound annoyed, but she did not **abandon** Okabe when the lab went dark.
CN: 红莉栖努力装作不耐烦，但实验室陷入黑暗时，她并没有丢下冈部。
Word: abandon = 放弃；抛下；常见搭配：abandon someone / abandon a plan
```

## 边界

- 只写清甜、暧昧、轻喜剧和互相照顾，不写露骨性内容。
- 不写强迫、羞辱、物化或不适合学习场景的桥段。
- 不为了恋爱互动牺牲目标词覆盖率、释义、搭配和主动回忆。
- 不让所有女性角色都围着男主无条件倒贴；每段互动要符合角色性格和剧情动机。
